## 🎯 一句话介绍

**虾逛**是一只基于 OpenClaw 打造的 AI 购物搭子，专为"不知道这件适不适合我"而设计。它看得懂衣服、记得住你的衣橱、比得了价格，还能帮你优雅脱身。

## 💡 解决什么问题

### 传统购物四大痛苦

- ❌ "这件怎么样？" → 闺蜜不在场，只能自己瞎猜
- ❌ "上次那件类似款" → 记忆模糊，全凭感觉
- ❌ "同款哪里便宜？" → 手动搜索，破坏逛街节奏
- ❌ "店员一直推，我不好意思拒绝" → 被强迫消费

### 虾逛的解决方案

- ✅ **实时陪伴** — 拍照→5秒出评价，就像带了不会累的闺蜜
- ✅ **跨次记忆** — 记住你买过的每一件，跨门店精准匹配
- ✅ **即时决策** — 全网比价+评价聚合，掏不掏钱不纠结
- ✅ **优雅脱身** — 提供防坑话术，保护你的钱包不被强迫消费

## 🛠️ 技术架构

```
用户（飞书/Telegram/微信）
        ↓
  OpenClaw Agent（虾逛大脑）
        ↓
   ┌─────────────┼─────────────┐
   ↓             ↓             ↓
images_understand  高德地图API   记忆系统
（视觉理解）    （附近导航）   （长期记忆）
```

**核心依赖：**

- OpenClaw Agent
- 视觉理解模型（images_understand）
- 高德地图 Web API（已配置 AMAP_KEY）
- 微信/飞书平台
