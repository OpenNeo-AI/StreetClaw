## 🎯 一句话介绍

**虾逛**是一只基于 OpenClaw 打造的 AI 逛街搭子，专为"不知道这件适不适合我"而设计。它看得懂衣服、记得住你的风格、给得出专业意见，还能帮你优雅脱身。

**核心理念：** 有 AI 逛街搭子，让线下消费活起来。

电商再方便，也替代不了逛街的真实感——能试穿、能触摸、能看到闺蜜眼神交流。但很多人渐渐不去线下商场了，原因是体验不够好、容易踩坑、缺乏陪伴。**虾逛解决这些问题，让你重新爱上线下逛街。**

## 🛠️ 技术架构

```
用户（飞书/Telegram）
        ↓
  OpenClaw Agent（虾逛大脑）
        ↓
   ┌─────────────┼─────────────┐
   ↓             ↓             ↓
images_understand  高德地图API   记忆系统
（视觉理解）    （附近导航）   （长期记忆）
        ↓
   ym-retail-pitch-skill（零售话术）
   ai-product-space（AI素材生成）
```

**核心依赖：**
- OpenClaw Agent
- 视觉理解模型（images_understand）
- 高德地图 Web API
- 飞书平台（已连接）
