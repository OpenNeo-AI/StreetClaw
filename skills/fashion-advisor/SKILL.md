# 穿搭参谋技能

## 激活场景
- 用户问"穿什么合适"
- 用户要去某个场合（通勤/约会/商务/休闲/运动）
- 用户问"这件适合我吗"
- 用户问"这个年龄/身材怎么穿"
- 用户问"全家出门怎么穿搭"

## 核心原则（所有人通用）
1. **说实话，不迎合**："这件不太适合" + 给原因
2. **场合优先**：不问风格，先问去哪、做什么
3. **质感 > 数量**：少买，买好的
4. **合身最重要**：肩线准、裤长对

---

## 一、男性穿搭公式

### 按年龄分

#### 20-28岁｜阳光大男孩
- 风格：街头、运动、休闲、少年感
- 必入：卫衣（灰/藏青/米白）、牛仔外套/工装夹克、球鞋/板鞋、束脚工装裤/直筒牛仔裤
- 避雷：oversized（合身最重要）、全身超过3色

#### 29-40岁｜轻熟质感期
- 风格：质感、简约、干练、Cityboy
- 必入：牛津纺衬衫、Blazer休闲西装（藏青/灰）、纯色针织衫、直筒西裤/垂感休闲裤、皮鞋（乐福/德比/切尔西）
- 加分：手表（简约款）、皮带颜色=鞋子颜色

#### 40-55岁｜成熟儒雅期
- 风格：经典、稳重、松弛感、老钱风
- 必入：高品质Polo衫（针织/丝光棉）、亚麻衬衫/西装、羊绒衫/开衫、精裁卡其裤/西裤
- 避雷：大logo、满身印花、裤子提到肚脐眼以上

### 男性场合公式

| 场合 | 公式 |
|------|------|
| 日常通勤 | 休闲衬衫（内搭白T）+ 直筒休闲裤 + 乐福鞋 |
| 周末约会 | 针织开衫 + 牛津纺 + 卡其裤 + 皮质乐福 |
| 商务会议 | Blazer + 衬衫 + 西裤 + 正装皮鞋；藏青色比黑更有品味 |
| 重要活动 | 深色西装外套款 + 深色裤子 + 棕色皮鞋 + 衬衫塞进去 |
| 兄弟聚会 | 设计感印花T/古巴领衬衫 + 黑色牛仔裤 + 切尔西靴 |
| 户外运动 | 速干T恤 + 短裤/压缩裤 + 跑鞋/徒步鞋 + 空顶帽 |

### 男性万能公式
- **颜色**：全身不超过3色 → 黑白灰+牛仔蓝/卡其色
- **合身**：改裤脚是性价比最高的变帅投资
- **质感**：少买快时尚，攒钱买好面料（羊毛/纯棉/亚麻/皮革）

## 二、女性穿搭公式

### 按人群分

#### 职场精英（25-40岁）｜飒爽与质感
- 投资高品质基础款：剪裁利落的西装、真丝衬衫、垂感阔腿裤
- 显贵小心机：同色系搭配（燕麦色西装+杏色内搭+白色阔腿裤）
- 细节加分：腕表、质感皮带、丝巾

#### 甜美学生党（18-25岁）｜元气与活力
- 叠穿大法：衬衫+毛衣马甲、T恤+牛仔背带裤
- 色彩魔法：低饱和度彩色（香芋紫/鹅黄）+ 白色下装
- 氛围感单品：百褶裙、帆布鞋、堆堆袜

#### 微胖梨形/苹果形身材｜扬长避短
- **梨形（上紧下松）**：修身/亮色上衣 + A字长裙/直筒伞兵裤藏肉
- **苹果形（打造腰线）**：V领上衣延伸颈部 + H型/A型连衣裙
- 万能法则：露出最细的部分（锁骨/脚踝），视觉瘦五斤

#### 小个子（158cm以下）｜视觉增高
- 高腰是命：上衣塞进去或选短款
- 下半身顺色：鞋子与裤子/裙子同色，腿长延伸
- 视觉重心上移：亮眼耳环/帽子，让人先看上半身

### 女性场合公式

| 场合 | 公式 |
|------|------|
| 日常通勤 | 有设计感衬衫（飘带/娃娃领）+ 九分烟管裤 + 乐福鞋 |
| 甜蜜约会 | 针织衫 + 缎面半身裙；小白裙/小黑裙 + 玛丽珍鞋 |
| 周末出游 | 工装裤 + 紧身T恤 + 运动鞋 + 棒球帽/墨镜 |
| 重要宴会 | 过膝连衣裙 + 高跟鞋 + 手拿包；避免全身大红/纯白 |
| 运动健身 | 支撑力好的运动内衣 + 速干leggings + 运动鞋 |

## 三、儿童穿搭公式

### 按年龄分

#### 婴幼儿（0-3岁）｜舒服第一
- A类面料、软底学步鞋
- 秋冬：连体衣（哈衣）+ 软底学步鞋
- 萌感加分：糖果色、可爱图案

#### 学龄前（3-6岁）｜活泼好动
- 耐磨弹性、方便自己穿脱
- 宽松卫衣/卡通T恤 + 运动裤/背带裤 + 魔术贴运动鞋
- 全身颜色不超过3个

#### 学龄儿童（6-12岁）｜校园mini潮人
- 校服之外有个性表达：棒球服/牛仔外套 + 连帽卫衣 + 牛仔裤/工装裤 + 板鞋
- 潮流启蒙：同色系穿搭或小配饰（棒球帽/小墨镜）

### 儿童场合公式

| 场合 | 男孩 | 女孩 |
|------|------|------|
| 日常上学 | 纯色卫衣 + 深色牛仔裤 + 板鞋 | 连衣裙/卫衣 + 运动裤 + 运动鞋 |
| 出游拍照 | 渔夫帽 + 牛仔外套 + 工装裤 | 碎花裙 + 打底裤 + 动物造型小背包 |
| 正式场合 | 小西装/格纹衬衫 + 卡其裤 | 公主裙 + 小皮鞋 + 白色袜子 |

## 四、全家出行穿搭协调

**核心原则：各自得体，彼此呼应，不必完全统一**
- 颜色协调：选定一个主色调，大人小孩各选该色调的单品
- 风格统一：商务休闲/运动休闲/街头风，全家同风格
- 场合优先：先确定最正式的那个人，其他人以此为准

---

## 五、避雷清单（所有人通用）
- ❌ 大logo、满身印花
- ❌ 过于紧身或过于宽松
- ❌ 裤子堆在鞋面上（改裤脚）
- ❌ 全身超过3个颜色
- ❌ 优衣库（江哥明确不买）

## 输出格式
1. 直接判断"适合/不适合"
2. 场合公式（具体到颜色+单品）
3. 一个关键细节
