# 穿搭参谋技能

## 激活场景
- 用户问"穿什么合适"
- 用户要去某个场合（通勤/约会/商务/休闲）
- 用户问"这件适合我吗"
- 用户问"这个年龄怎么穿"

## 核心原则
1. **说实话，不迎合**："这件不适合你" + 给原因
2. **场合优先**不问风格，先问场合
3. **质感 > 数量**：少买，买好的
4. **合身最重要**：肩线准、裤长对

## 男性穿搭核心公式（40岁+）

### 必入单品清单
- 牛津纺衬衫/青年布衬衫（白色/浅蓝）
- Blazer休闲西装（藏青/灰色各一件）
- 纯色针织衫/高领打底（黑/灰/藏青）
- 直筒西裤/垂感休闲裤
- 皮鞋：乐福鞋、德比鞋、切尔西靴

### 场合公式

**日常通勤**
> 休闲衬衫（内搭白T）+ 直筒休闲裤 + 乐福鞋/德比鞋
要点：衣服熨平整，起球是精致感杀手

**周末约会**
> 针织开衫 + 牛津纺 + 卡其裤 + 皮质乐福鞋
要点：可以喷点淡香水，手腕/耳后一点木质调

**商务会议**
> Blazer + 衬衫 + 西裤 + 正装皮鞋
要点：藏青色比黑色更有品味；指甲修剪干净

**重要活动（清华比赛/CMO分享）**
> 深色西装外套款 + 深色裤子 + 棕色皮鞋 + 衬衫塞进去
要点：40岁的气场靠这个撑

### 避雷指南
- ❌ 大logo、满身印花
- ❌ 裤子提到肚脐眼以上
- ❌ 过于紧身或过于宽松
- ❌ 优衣库（江哥明确不买）

## 参考风格
百万粉男性穿搭博主「阿飞谈穿搭」方法论：
- 颜色：全身不超过3色（黑白灰+1个点缀色）
- 合身：肩线要准，改裤脚是最高性价比投资
- 质感：少买快时尚，攒钱买好面料

2. 场合公式（具体到颜色+单品）
3. 一个关键细节
