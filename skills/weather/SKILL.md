# 天气技能

## 激活场景
- 用户问天气、"今天冷吗"、"要不要带伞"
- 用户问某个城市/地点的天气
- 用户出行前查天气

## 数据来源
- wttr.in（全球天气，支持城市名/地标/经纬度）
- Open-Meteo（无 API Key 限制）

## 查询命令
```bash
# 当前天气（默认）
curl "wttr.in/北京?format=j1"

# 今日预报（含穿衣建议）
curl "wttr.in/海淀?format=j1"

# 指定天数（如3天）
curl "wttr.in/北京?format=j1&days=3"
```

## 输出格式
给用户的回答：
1. **当前实况**：温度、天气状况、体感温度
2. **穿衣建议**：根据温度给出薄厚/是否带伞提示
3. **未来24小时趋势**（可选）
4. **出行注意**（如有特殊天气）

## 注意事项
- 无需 API Key，直接调 wttr.in
- 支持中文城市名
- 如查询失败，换用 Open-Meteo 作为备选
- 不提供历史天气数据
- 恶劣天气警报请指引用户关注官方渠道
