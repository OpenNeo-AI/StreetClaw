## 一、产品概述

### 1.1 是什么

**虾逛**是基于 OpenClaw 框架打造的 AI 逛街搭子，通过飞书平台与用户实时连接，提供指路、比价、参谋、砍价、情绪价值等全链路陪伴服务。

### 1.2 解决什么问题

| 痛点 | 传统解法 | 虾逛的解法 |
|------|----------|-----------|
| 这件适合我吗？ | 靠感觉/闺蜜不在 | 拍照→AI即时评价 |
| 上次那件类似款？ | 记忆模糊 | 跨门店记住衣橱历史 |
| 同款哪里便宜？ | 手动搜索 | 全网比价+优惠推送 |
| 店员强推不好意思拒绝 | 硬着头皮买 | AI防坑话术优雅脱身 |
| 逛太久累了 | 中断行程 | AI持续陪伴不抱怨 |

### 1.3 目标用户

- 都市年轻人（22-40岁），热爱逛街但时间有限
- 购物时缺乏第二意见，需要靠谱的"在场感"
- 希望买得值、买得爽、不踩坑

## 二、核心功能

### 2.1 已上线功能

| 功能模块 | 功能名称 | 详细描述 | 技术实现 |
|----------|----------|----------|----------|
| 视觉识别 | 慧眼识物 | 拍照商品，获得穿搭评分+同款全网比价 | images_understand |
| 导航 | 附近导航 | 高德地图API，附近门店路线 | 高德Web API |
| 记忆 | 衣橱历史 | 跨门店记住购买记录，精准推荐 | OpenClaw记忆系统 |
| 比价 | 全网比价 | 线上+本地商品价格对比 | shopping-expert |
| 话术 | 砍价助攻 | 提供砍价话术，帮你喊出心理价 | ym-retail-pitch-skill |
| 话术 | 防坑话术 | 店员强推时，优雅脱身 | 内置话术库 |
| 情绪 | 情绪充电 | 购物受挫时，提供安慰和信心 | SOUL.md人格系统 |
| 内容 | 种草文案 | AI改写文案，朋友圈/小红书种草 | humanizer |
| 内容 | 每日热榜 | 推送当日热门购物攻略 | daily-hot-push |

### 2.2 规划中功能

- 试衣评分：多件试穿拍照，AI对比打分
- 主动提醒：护肤/服装快用完时主动发起复购
- 商场实时折扣：主动推送商场/品牌优惠

## 三、技术架构

### 3.1 系统架构图

```
                    ┌─────────────────┐
                    │   用户（江哥）   │
                    └────────┬────────┘
                             │ 飞书消息
                             ▼
               ┌────────────────────────┐
               │    OpenClaw Gateway     │
               │  (飞书 WebSocket 长连接) │
               └────────────┬───────────┘
                            │ Agent 大脑
          ┌─────────────────┼─────────────────┐
          ▼                 ▼                 ▼
  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
  │视觉理解模块   │  │ 导航模块     │  │  记忆模块     │
  │images_       │  │ amap-       │  │ self-        │
  │understand    │  │ navigator   │  │ improving    │
  └──────────────┘  └──────────────┘  └──────────────┘
          │                 │                 │
          ▼                 ▼                 ▼
  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
  │shopping-      │  │ 高德地图API  │  │  知识图谱    │
  │expert        │  │             │  │  ontology    │
  │比价/优惠券   │  │ 附近门店    │  │             │
  └──────────────┘  └──────────────┘  └──────────────┘
```

### 3.2 技术选型

| 组件 | 技术 | 说明 |
|------|------|------|
| Agent框架 | OpenClaw | 核心引擎 |
| 平台连接 | 飞书 WebSocket | 长连接，无需公网地址 |
| 视觉理解 | MiniMax images_understand | 拍照识物 |
| 地图导航 | 高德地图 Web API | 已配置AMAP_KEY |
| 记忆系统 | OpenClaw Memory + ontology | 长期+结构化记忆 |
| 技能管理 | SkillHub | 40+技能即插即用 |

### 3.3 平台接入

```
飞书 Open Platform
    ├── 创建企业自建应用
    ├── 配置权限（im:message 等）
    ├── 开启机器人能力
    ├── 事件订阅（WebSocket 长连接）
    └── 发布应用
        ↓
    OpenClaw Gateway（处理所有消息）
        ↓
    虾逛 Agent（理解→决策→执行）
```
